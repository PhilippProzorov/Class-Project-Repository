# Вариант 12 -- адресная книга

Хранит список адресов различных людей. Адрес состоит из улицы, номера 
дома и номера квартиры. Человек задаётся фамилией, для каждого человека 
хранится лишь один адрес.

Методы: добавление пары человек-адрес, удаление человека, изменение 
адреса человека, получение адреса человека, получение списка людей,
живущих на заданной улице или в заданном доме.
